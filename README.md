# do_it_app

A simple Python command-line to-do list app where you can add, view, and remove tasks.

## Features:
- Add tasks
- Remove tasks
- View all tasks

## How to Run:
1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/ricoswabii/do_it_app.git
   cd do_it_app
   ```

2. Run the script:

   ```bash
   python doitnow.py
   ```
